Download And Installation

For first you need to download unicentapos , then create database and use mysql databased, then you can download this source code project, run xampp server for installation on localhost, create new fodlder and name it with your own, on C:/Xampp/htdocs/projectfolder then extract all files download on your project folder , need to configure on connection folder, just integration and change with your mysql databased and save it, now your unicenta web app is integration with your databased , then we must installed unicenta dashboard backoffice web apps , need to install node npm just download and install npm , then install angular globaly on your device , open npm node then run command npm install -g @angular/cli after installation angular you can open project folder then we work with node npm again just open node shell terminal or cmd then run command cd C:/Xampp/htdocs/projectfolder and run command npm install && ng serve --o and open localhost:4200 on web browser. Now your unicentapos with angular is ready to live.

How to online this web apps ?? you need to create repo on github and then integration with heroku / vercel host, or you can push and deploy on firebase .Just register on github / gitlab / bitbucket then create new repo and upload this source code on your repo , after push upload repo success you need to deploy it on heroku just register on heroku, create new app, and connected with your repo,and deploy it... congratulation your web app is online now.
